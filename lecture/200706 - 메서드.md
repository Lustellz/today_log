메서드(함수)를 만드는 이유: 중복되는 코드를 줄여 재활용성을 높이기 위함.

메서드 작성 방법

```java
static returnType methodName(dataType parameters){
    //Method scope
    return type;
}
```

- 원하는 결과값을 알아야지 메서드의 리턴형을 정할 수 있음(데이터 형의 일치가 중요)

- parameters=인자=매개변수(=arguments)
